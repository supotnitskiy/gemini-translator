##  Установка

клонировать это репозиторий

Создать в клонированной папке каталог GeminiTranslator , во внутрь которого скопировать файлы программы.

создать файл .env  в котором прописать эти параметры

VNC_PASSWORD=ВАШ СЕКРТЕНЫЙ ПАРОЛЬ ДЛЯ VNC
APP_DIR=GeminiTranslator

потом docker compose up -d

и подключаетесь по vnc по 10000 порту к серверу где вы развернули.
